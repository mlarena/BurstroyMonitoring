# Руководство пользователя: Сводные отчеты

Модуль «Сводные отчеты» предназначен для анализа данных, полученных со всех датчиков постов мониторинга за выбранный период времени. Он позволяет объединить показатели температуры, влажности, состояния дорожного покрытия и других параметров в единую удобную таблицу.

## Как сформировать отчет

1. **Выбор объекта**: В верхней части страницы выберите нужный «Пост мониторинга» из выпадающего списка.
2. **Настройка периода**:
   - В поле **«Дата начала»** выберите день и время, с которых нужно начать отчет.
   - В поле **«Дата окончания»** выберите конечную точку периода.
3. **Выбор детализации (Интервал)**:
   - **Без группировки**: Вы увидите каждое отдельное измерение, которое прислал датчик.
   - **10 / 30 / 60 мин**: Система рассчитает средние значения за указанные промежутки времени. Это удобно для оценки динамики за длительный период (например, за неделю или месяц).
4. **Просмотр**: Нажмите кнопку **«Сформировать сводный отчет»**.

## Работа с таблицей отчета

- **Заголовки**: Таблица разделена на группы по датчикам. В верхней строке указано название датчика и его серийный номер, в нижней — конкретный параметр (например, «Температура воздуха»).
- **Прокрутка**: Если данных много и таблица не помещается по ширине, используйте полосы прокрутки. Для вашего удобства они расположены как **снизу**, так и **сверху** таблицы.
- **Пустые значения**: Если в ячейке стоит прочерк «-», это означает, что в данный момент времени датчик не передавал данные.

## Экспорт данных

Вы можете сохранить сформированный отчет на свой компьютер в двух форматах:
- **Excel (зеленая кнопка)**: Создает полноценный документ с красивым оформлением и сгруппированными заголовками. Идеально подходит для печати и официальных отчетов.
- **CSV (серая кнопка)**: Сохраняет данные в текстовом формате. Удобно для последующей загрузки в другие системы анализа данных.

---
*Примечание: Если после нажатия кнопки «Сформировать» появилось сообщение «Нет данных за выбранный период», попробуйте выбрать другой диапазон дат или проверить, активен ли данный пост мониторинга.*
